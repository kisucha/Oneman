# 🎯 공동 목표 (Company Goals)

_이 파일은 **모든 에이전트가 매번 읽는** 회사의 북극성입니다. 자유롭게 편집하세요._

## 장기 목표 (1년)
- [ ] 유튜브 구독자 10만 달성
- [ ] 초급/중급/고급 영단어 시리즈 각 60편 완성 (총 180편, 5400단어)
- [ ] 배치 자동화 파이프라인으로 레벨별 전체 영상 일괄 생성

## 단기 목표 (1개월)
- [ ] 샘플 1편 완성 (30단어, 원하는 포맷/품질 확인)
- [ ] 초급 영단어 1800개 목록 확보 (AI 선정 또는 공개 단어장)
- [ ] 초급 60편 배치 자동 생성 파이프라인 완성

## 콘텐츠 규격 (에이전트 공통 기준)
- 단어 수: 30개 / 편
- 슬라이드 포맷: 단어 + IPA + 한국어 뜻 + 예문
- 나레이션: 단어 → 예문 → pause(2초) → 한국어 뜻 → pause(3초) 구조
- 영상 길이 목표: 5~8분 (30단어 기준)
- 파일명 규칙: beginner_001_apple.mp4 형식
