# 🏢 회사 정체성 / 톤앤매너

_브랜드 보이스, 톤, 절대 금지어 등을 적으세요. 모든 에이전트가 매번 참조합니다._

- **회사 이름:** CHK Contents
- **한 줄 소개:** 자기개발 컨텐츠 만드는 1인 기획사
- **채널 상태:** 신생 채널 — 초기 콘텐츠 구축 단계
- **대표자:** 대표님 (1인 운영)
- **타깃 청중:** 연령 제한 없음. 왕초보부터 시작해서 꾸준히 보면 실력이 오르는 구조. 시험/취업보다 순수한 학습 목적. 조용한 공간에서 집중하거나 배경음처럼 트는 ASMR형 시청자.
- **핵심 가치:** 압박감 없는 학습, 천천히 쌓이는 실력, 누구나 접근 가능
- **브랜드 톤:** 차분하고 느린 속도. 강의가 아닌 '같이 공부하는 느낌'. 단순하고 깔끔한 슬라이드. 어두운 배경으로 눈이 편한 화면. 음악 없거나 매우 조용한 배경음.
- **금기 (절대 하지 말 것):** 빠른 속도·압박감 있는 나레이션 금지. 광고성·낚시성 제목 금지. 과도한 화면 전환·팝업·효과음 금지. 시험 점수 중심 스트레스 유발 금지. 복잡한 문법 설명으로 시작 금지.

## 콘텐츠 방향

- 왕초보 기준으로 시작해서 레벨을 천천히 올리는 시리즈 구성
- 한 영상에 한 주제 집중 (단어 또는 표현 또는 발음)
- 반복 노출 구조: pause → 생각할 시간 → 정답 확인
- 한국인이 자주 헷갈리는 포인트 집중 설명
