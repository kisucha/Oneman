## 요약
중급 수준의 영어 단어 20개를 학습하는 콘텐츠 제작. 나레이션은 영어만 읽고 TTS 지시어는 분리하여 스크립트 작성 후 영상 조립.

## 분배
- **🔍 Researcher**: 중급 수준의 영어 단어 20개를 선정하고 각 단어의 발음 기호, 한국어 뜻, 일상적인 예문을 포함한 데이터를 수집하여 다음 에이전트에게 전달할 수 있도록 파일에 저장하세요.
- **✍️ Writer**: 선정된 20개 단어로 나레이션 스크립트를 작성하세요. 각 단어는 4번 반복하고 2초 간격으로 발음하며, 예문은 3초 후 읽도록 구성. TTS 지시어(···Xs···)와 나레이션 내용을 분리하여 UTF-8 인코딩으로 script.txt 파일 저장.
- **🎨 Designer**: 20개 단어 학습 썸네일을 제작하세요. 빨강 배경에 Pretendard Bold 폰트 사용, 'Vocabulary Training — Level 2'와 'Today's 20 Words' 텍스트 중앙 배치, 폰트 경로 정확히 지정하여 생성.
- **✂️ Editor**: script.txt를 기반으로 kokoro-onnx TTS로 나레이션 WAV 파일 생성 후 FFmpeg로 영상 조립. 각 단어 슬라이드는 '단어 : 뜻' 형식, 예문 슬라이드는 영문/한글 병기. final.mp4 파일 생성 전 음성/영상 존재 여부 검증.
- **📱 Secretary**: 생성된 final.mp4와 썸네일의 품질을 검증하고 최종 업로드를 준비하세요. 각 단계별 파일 존재 여부 확인 후 게이트키핑 프로세스 적용.
