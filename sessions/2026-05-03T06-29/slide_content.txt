Vocabulary Training — Level 2
Today's 20 Words

1. Analyze ( /ˈænəlaɪz/ )
분석하다
We need to analyze the data carefully.
데이터를 신중하게 분석해야 합니다.

2. Approach ( /əˈproʊtʃ/ )
접근하다, 방법
Let's take a different approach to solve this problem.
이 문제를 해결하기 위해 다른 접근 방식을 사용해 봅시다.

3. Benefit ( /ˈbenɪfɪt/ )
이익, 혜택
Regular exercise has many health benefits.
규칙적인 운동은 많은 건강상의 이점이 있습니다.

4. Concept ( /ˈkɑːnsept/ )
개념, 생각
The concept of time management is important.
시간 관리의 개념은 중요합니다.

5. Context ( /ˈkɑːntekst/ )
맥락, 문맥
You need to understand the context of the situation.
상황의 맥락을 이해해야 합니다.

6. Define ( /dɪˈfaɪn/ )
정의하다
Can you define this term for me?
이 용어를 정의해 줄 수 있나요?

7. Environment ( /ɪnˈvaɪrənmənt/ )
환경
We must protect our environment for future generations.
미래 세대를 위해 우리의 환경을 보호해야 합니다.

8. Feature ( /ˈfiːtʃər/ )
특징, 기능
The main feature of this phone is its camera.
이 휴대폰의 주요 특징은 카메라입니다.

9. Function ( /ˈfʌŋkʃn/ )
기능, 작용
What is the function of this button?
이 버튼의 기능은 무엇입니까?

10. Identify ( /aɪˈdentɪfaɪ/ )
식별하다, 확인하다
Can you identify the problem in this equation?
이 방정식의 문제를 식별할 수 있나요?

11. Impact ( /ˈɪmpækt/ )
영향, 충격
The pandemic had a huge impact on the economy.
팬데믹은 경제에 큰 영향을 미쳤습니다.

12. Interpret ( /ɪnˈtɜːrprət/ )
해석하다, 통역하다
It's important to interpret the data correctly.
데이터를 올바르게 해석하는 것이 중요합니다.

13. Justify ( /ˈdʒʌstɪfaɪ/ )
정당화하다
Can you justify your decision to the board?
이사회에게 당신의 결정을 정당화할 수 있나요?

14. Method ( /ˈmeθəd/ )
방법, 방식
This method has proven to be very effective.
이 방법은 매우 효과적인 것으로 입증되었습니다.

15. Neglect ( /nɪˈɡlekt/ )
무시하다, 소홀히 하다
Don't neglect your responsibilities.
당신의 책임을 소홀히 하지 마세요.

16. Obtain ( /əbˈteɪn/ )
얻다, 획득하다
You need to obtain a visa before traveling.
여행 전 비자를 발급받아야 합니다.

17. Perceive ( /pərˈsiːv/ )
지각하다, 인식하다
I perceive a change in his attitude.
그의 태도 변화를 느낍니다.

18. Quality ( /ˈkwɑːləti/ )
품질, 질
Quality is more important than quantity.
양보다 질이 더 중요합니다.

19. Resolve ( /rɪˈzɑːlv/ )
해결하다, 결심하다
We need to resolve this issue immediately.
우리는 이 문제를 즉시 해결해야 합니다.

20. Strategy ( /ˈstrætədʒi/ )
전략, 전술
Our marketing strategy needs improvement.
우리의 마케팅 전략은 개선이 필요합니다.
