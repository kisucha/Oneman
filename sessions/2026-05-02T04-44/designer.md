## 🎨 브랜드 아이덴티티 및 채널 비주얼 방향 설계

### 1. 브랜드 디자인 브리프 (Brand Design Brief)

#### ▶ 브랜드 페르소나
CHK Contents는 **영어 학습을 쉽고 재미있게** 만들어주는 친절한 튜터 타입의 브랜드입니다.

| 항목 | 내용 |
|------|------|
| 브랜드 성격 | 친근하고, 도움을 주는, 실용적인 |
| 브랜드 톤앤매너 | 가볍지만 전문적인 / 격식 없지만 신뢰감 있는 |
| 타겟 오디언스 | 초보~중급 자가학습자 (특히 20~30대 한국인) |
| 브랜드 핵심 가치 | 실용성 · 반복성 · 접근성 · 즐거움 |

### 2. 컬러 시스템 (Color System)

#### ▶ 메인 컬러
- **Primary Color:** `#4A90E2` (신뢰감 있는 파랑, 학습의 느낌)
- **Secondary Color:** `#FFD43B` (밝은 노랑, 에너지와 긍정)

#### ▶ 서포트 컬러
- `#F0F4F8` (배경용 밝은 그레이)
- `#2C3E50` (텍스트/헤딩용 다크 블루 그레이)
- `#E74C3C` (주의/강조용 레드)

#### ▶ 컬러 심볼리즘
| 색상 | 역할 | 감정 |
|------|------|------|
| 파랑 (#4A90E2) | 핵심/신뢰 | 안정감, 학습 |
| 노랑 (#FFD43B) | 강조/CTA | 활기, 집중 |
| 그레이 (#F0F4F8) | 배경 | 클린, 여유 |
| 레드 (#E74C3C) | 경고/퀴즈 | 흥미, 도전 |

## ✅ 썸네일 컨셉 3안 (Thumbnail Concepts)

### Concept A: **“표현 중심 캐주얼 썸네일”**
- **배경색:** 메인 블루 `#4A90E2`
- **중앙 이미지:** 감정 있는 캐릭터가 큰 말풍선으로 표현을 말함
- **상단 텍스트:** "오늘의 표현!" (굵은 폰트)
- **하단 텍스트:** "That's a wrap!" (표현 강조)
- **장점:** 학습자가 즉시 무엇을 배우는지 파악 가능

### Concept B: **“숏폼 콘텐츠 리액션 썸네일”**
- **배경색:** 밝은 노랑 `#FFD43B`
- **중앙 이미지:** 놀란/재미있는 얼굴 + “???” 말풍선
- **하단 텍스트:** “이거 영어로 뭐라고 해?” (궁금증 유도)
- **장점:** 릴스/쇼츠 형태에 적합, 클릭률 ↑

### Concept C: **“뉴스 기반 회화 썸네일”**
- **배경색:** 뉴스 감성의 회색 `#F0F4F8`
- **중앙 이미지:** TV 스크린 위에 영상 썸네일 느낌
- **상단 텍스트:** “Weekly News Talk”
- **하단 텍스트:** “오늘은 어떤 주제?”
- **장점:** 정보성과 실용성 강조, 시청자 기대감 유도

### 5. 비주얼 시스템 가이드 요약 (Visual System Guide Summary)

| 항목 | 가이드라인 |
|------|------------|
| 썸네일 크기 | 1280×720px (YouTube 기준) |
| 폰트 | Poppins (Eng) / Noto Sans KR (Kor) |
| 메인 컬러 | `#4A90E2` + `#FFD43B` |
| 감정 이미지 | 과장되지만 자연스러운 리액션 (웃음, 놀람, 궁금함) |
| 말풍선 스타일 | 단순하고 굵은 외곽선 + 흰색 배경 |
| CTA 버튼 | 노랑 배경 + 검정 텍스트 (Ex: “지금 보기”, “PDF 받기”) |
