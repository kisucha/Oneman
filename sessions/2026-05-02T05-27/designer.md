# 🎨 CHK Contents 썸네일 및 템플릿 디자인 제안서

## 1. 🎯 브랜드 디자인 브리프

### 🔷 브랜드 컬러
- **파랑 (#3B82F6)** — 신뢰, 학습, 클리어함 상징  
- **노랑 (#FBBF24)** — 활기, 집중, 가독성 강조  
- **보조 색상**: 밝은 그레이(#F3F4F6), 흰색(#FFFFFF)

### 🔠 타이포그래피
- **헤드라인용**: `Pretendard ExtraBold` (굵고 명확하게)
- **내레이션/자막용**: `Poppins Regular` or `Noto Sans KR Medium` (가독성 높은 무난한 선택)

### 🖼 레퍼런스
- 유튜브 쇼트폼 채널 중 “bright, minimal, playful” 느낌을 살린 디자인  
- 예시: “English with Lucy Shorts”, “Learn English with Emma”

## 2. 🎞 썸네일 컨셉 3안

### ✅ 기본 템플릿 구조:
- 배경: 파랑 + 노랑 그라데이션 또는 블록 컬러
- 중앙 텍스트: “30 Words”, “#1”, “Everyday Verbs!”
- 아이콘/이미지: 단어와 관련된 간단한 일러스트 (e.g., ‘run’ → 달리는 사람)

---

### 🟦 [컨셉 A] 강조형 텍스트 + 일러스트
- 배경: 파랑 (#3B82F6)  
- 텍스트: 노랑 (#FBBF24)  
- 요소: 텍스트 위에 간단한 일러스트 추가 (e.g., 책, 연필 등)  
- 폰트: Pretendard ExtraBold  
- 예시:  
  ```
  30 Words
  Everyday Verbs!
  #1
  ```

### 🟨 [컨셉 B] 테두리형 블록 + 단어 힌트
- 배경: 노랑 (#FBBF24)  
- 텍스트: 파랑 (#3B82F6)  
- 테두리: 검은색 실선으로 감싸 강한 시선 유도  
- 요소: 중앙에 “Verb”라는 단어 카테고리 + 관련 이모지 (🏃, 🍽️ 등)  
- 예시:  
  ```
  Verb #1
  Run /rʌn/
  ```

### 🟪 [컨셉 C] 카드형 레이아웃 + 번호 시리즈
- 배경: 파랑 (#3B82F6)  
- 카드: 흰색 배경 (약간 라운드 처리)  
- 요소: 카드 안에 큰 번호 (#1), 카테고리 (“Verbs”), 부제 (“Beginner Friendly”)  
- 예시:  
  ```
  #1
  Verbs
  Beginner Friendly
  ```

## 3. 🧾 비주얼 시스템

### ✅ 전체 템플릿 구성 (Figma 기준)
1. **썸네일 템플릿** (A, B, C 모두 제작)
2. **오프닝 카드** (e.g., “Let’s Learn 30 Words!”)
3. **단어 카드 템플릿** (발음/뜻/예문 포함)
4. **아웃트로 템플릿** (“Subscribe for More!”)

### 🎨 디자인 요소 가이드
| 요소 | 스타일 |
|------|--------|
| 배경 | 파랑 + 노랑 (선택에 따라 조합) |
| 버튼/CTA | 테두리 없는 둥근 버튼, 텍스트는 반전 색상 |
| 글꼴 | Pretendard / Poppins / Noto Sans KR |
| 일러스트 | 픽토그램 스타일, 단순하고 친근한 이미지 |

## 4. 📐 디자인 가이드 요약 (Figma Style Guide 기반)
