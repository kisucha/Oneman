### 🧰 3. Canva 무료 버전 내 실제 제작 방법

#### A. 브랜드 색상 등록 방법

1. Canva 접속 → 좌측 상단 ‘+’ 클릭 → “Custom Size” 선택 (예: 1080x1920 – 세로형 쇼츠)
2. 우측 ‘Brand Kit’ 클릭
3. ‘Colors’ 탭 → ‘Add new color’로 파랑(#007BFF)과 노랑(#FFD400) 등록

#### B. 썸네일 템플릿 제작

1. 위에서 만든 사이즈로 새 디자인 시작
2. 배경색을 파랑으로 지정
3. 중앙 상단에 노랑 바 형태 배경 추가
4. 텍스트 박스 추가:
   - 상단: ‘Ready to learn?’ (흰색, 굵은 글씨)
   - 중앙: 단어 예시 (예: “Apple”, “Book”) - Pretendard 유사 무료 폰트 사용
5. ‘Templates’ > ‘Save as template’ 클릭하여 저장

#### C. 템플릿 재사용

- 다음 제작 시: ‘Templates’ > ‘My templates’에서 선택 후 복제
- 텍스트만 변경하여 신속하게 제작 가능

### ⚠️ 4. 제한사항 및 대응 방안

| 제한사항 | 대응 방안 |
|----------|-----------|
| 커스텀 폰트 등록 불가 | Pretendard와 유사한 무료 폰트(Spoqa Han Sans 등) 사용 |
| 팀원 초대 및 협업 제한 | 제작 템플릿을 공유링크로 배포 후 수정 요청 |
| 저장 가능한 색상 수 제한(최대 5개) | 브랜드 컬러 + 강조 컬러 중심으로 제한 선에서 활용 |
