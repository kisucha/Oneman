### 🎬 2. CapCut 기반 숏폼 제작 워크플로우 (무료 기준)

#### Step 1: 영상 촬영 or 스톡 영상 준비

- **촬영**: 스마트폰 카메라 또는 CapCut 내장 녹화 기능 활용
- **스톡 영상**: 무료 스톡 영상 제공 사이트(Pixabay, Pexels 등) 사용

#### Step 2: CapCut에서 프로젝트 시작

1. CapCut 앱 열기 → ‘+’ 버튼으로 새 프로젝트 생성
2. 영상 파일 추가
3. 타임라인 편집: 필요 구간 자르기, 속도 조절, 트랜지션 추가

#### Step 3: 자동 자막 생성

1. 타임라인 하단 ‘텍스트’ → ‘자동 자막’ 선택
2. 언어 설정: English → Korean (또는 반대로)
3. 자막 스타일 커스텀 (폰트, 위치, 색상 등)

#### Step 4: 텍스트 및 이펙트 삽입

- 단어별 강조를 위한 텍스트 박스 추가
- 단어 발음/뜻을 나타내는 간단한 애니메이션 템플릿 적용
- CapCut 내장 이펙트(줌인/아웃, 페이드 등) 활용

#### Step 5: 음원 추가

- CapCut 음악 라이브러리에서 무료 BGM 선택
- 음량 조절 및 페이드 인/아웃 설정

#### Step 6: 내보내기 및 저장

- 해상도: 1080x1920 (세로형)
- 포맷: MP4 (기본값)
- SNS 바로 업로드 옵션 활용 가능

### 🔁 3. 자동화 가능 포인트

| 항목 | 자동화 가능 여부 | 설명 |
|------|------------------|------|
| 자동 자막 생성 | ✅ 가능 | 영상 업로드 후 1회 클릭으로 자동 생성 |
| 텍스트 템플릿 | ⚠️ 반자동 | 기본 템플릿은 제공되나, 내용 입력은 수동 |
| 자막 스타일 적용 | ⚠️ 반자동 | 스타일 템플릿은 저장 후 재사용 가능 |
| 썸네일 자동 생성 | ❌ 불가 | 별도 도구(Canva 등) 필요 |

📌 *자동화 제안*:
- 매 편 자막 생성 → 자동 → 이후 수동 보정만 진행
- 단어별 텍스트 박스는 템플릿으로 저장하여 재사용
- 자막 스타일은 일관성 있게 1~2가지로 고정하여 편집 시간 단축
