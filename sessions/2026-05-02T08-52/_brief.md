# 📋 작업 브리프

## 요약
제안된 콘텐츠 제작 방안이 실제로 무료로 구현 가능한지 검토하고, 각 파트별 도구와 운용 방법을 정리

## 분배
- **🔍 Researcher**: 제안된 툴(도구)들이 현재 무료로 사용 가능한지 여부와 각 도구의 상세 스펙 및 한계점 분석
- **✍️ Writer**: 무료 도구 기반으로 자동화 가능한 스크립트 작성 방식 및 프롬프트 구조 재검토
- **🎨 Designer**: Canva 무료 버전 내에서 브랜드 템플릿 제작이 가능한지, 실제 제작 방법 및 제한사항 정리
- **✂️ Editor**: CapCut 무료 버전을 이용한 숏폼 편집 워크플로우 작성 및 자동화 포인트 확인
- **💻 Developer**: Make.com 무료 플랜 내에서 ChatGPT API 기반 자동화 파이프라인 구축 가능성 평가 및 제한사항 정리
