## 🎬 초기 콘텐츠 라인업 & 제작 일정 (1~3개월 기준)

### 🎯 목표
- **MVP 영상 12개 제작**

---

## 🗓️ 제작 타임라인 (4주 단위)

### 📆 **Week 1**
- **콘텐츠 주제**: *“Hello, How are you?” – 초급 인사말 익히기*
- **영상 길이**: 3분 이내
- **스크립트 작성**: 완료
- **녹음 + 편집**: 1일
- **썸네일 디자인**: Canva 템플릿 사용
- **업로드**: 월요일 오전 9시 (최적 시간)

### 📆 **Week 2**
- **콘텐츠 주제**: *“Be 동사의 기본” – am, is, are*
- **영상 길이**: 4분 이내
- **스크립트 작성**: 완료
- **녹음 + 편집**: 1일
- **썸네일 디자인**: 파랑+노랑 배경 템플릿
- **업로드**: 월요일 오전 9시

### 📆 **Week 3**
- **콘텐츠 주제**: *“What’s your name?” – 자기소개 회화*
- **영상 길이**: 3분 30초
- **스크립트 작성**: 완료
- **녹음 + 편집**: 1일
- **썸네일 디자인**: 템플릿 재사용 + 얼굴 아이소レーション
- **업로드**: 월요일 오전 9시

### 📆 **Week 4**
- **콘텐츠 주제**: *“My Daily Routine” – 일상 표현 배우기*
- **영상 길이**: 5분 이내
- **스크립트 작성**: 완료
- **녹음 + 편집**: 1일
- **썸네일 디자인**: 템플릿 + 시각적 일과 요소 삽입
- **업로드**: 월요일 오전 9시

## 📋 반복 작업 플로우

| 단계 | 내용 | 소요 시간 |
|------|------|------------|
| 1. 주제 선정 | 공통 니즈 기반 | 1시간 |
| 2. 스크립트 작성 | 초급 수준 중심 | 2시간 |
| 3. 녹음 | 클린 녹음 확보 | 1시간 |
| 4. 편집 | CapCut 사용 | 2시간 |
| 5. 썸네일 | Canva 템플릿 | 30분 |
| 6. 업로드 | 제목, 태그, 설명 입력 | 30분 |

## 📌 제작 시 유의사항

- **오프닝/엔딩 통일**: 짧고 강렬한 인트로 5초 + 브랜드 문구
- **자막 필수**: 접근성 강화 및 시청 유지율 증대
- **댓글 유도 문구**: “가장 기억에 남는 인사말은 무엇인가요?” 등
- **영상 내 CTA(Call To Action)**: 좋아요, 구독, 알림설정 권유 포함
