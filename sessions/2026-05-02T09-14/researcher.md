## ✅ Google Sheets, ChatGPT, Notion, Make.com 무료 플랜 자동화 가능 범위 및 제한사항 분석

### 1. **Google Sheets (무료 플랜)**  
**기본 제한사항**
- 최대 500개의 동시 편집자
- 업로드 용량: 최대 100MB
- 스크립트 실행 시간: 하루 90분 (무료 사용자 기준)
- API 호출 제한: 하루 약 100~200회 (IP 기준 다이나믹 제한)

**자동화 가능 범위**
- Google Apps Script를 통한 간단한 트리거 기반 자동화 (예: 폼 제출 후 자동 칼럼 업데이트, 메일 전송 등)
- Google Forms와 연동하여 실시간 데이터 수집
- Make.com 등과 연동하여 외부 서비스 데이터 자동 수집/갱신 가능

**제한사항**
- 복잡한 실시간 처리 불가 (예: AI 기반 데이터 분류)
- 고속 API 연동 부족으로 대용량 실시간 처리 불가

---

### 2. **ChatGPT (무료 플랜)**  
**기본 제한사항**
- 일일 사용량 제한 존재 (정확한 수치는 OpenAI 정책에 따라 변동)
- 고정 IP 대역에서 접근 제한 가능성 있음
- 고급 기능(GPT-4 이상) 일부 제한

**자동화 가능 범위**
- Make.com과의 연동을 통해 입력 텍스트에 대한 GPT 응답 자동화
- 간단한 질의응답, 요약, 템플릿 생성 자동화 가능

**제한사항**
- 일일 사용 제한으로 인한 반복 대용량 처리 어려움
- 고급 기능 사용 불가 → 브랜드 톤 유지/복잡한 스크립트 생성 등에 제약

---

### 3. **Notion (무료 플랜)**  
**기본 제한사항**
- 최대 1000개의 블록/페이지
- 무제한 멤버 초대 가능하나, 동시 편집자는 최대 5명까지 제한
- API 호출 제한: 하루 1000회 미만 (계정당)

**자동화 가능 범위**
- Make.com과 연동하여 페이지/데이터베이스 자동 업데이트
- 간단한 스크립트 또는 Zapier 연동으로 일정/할일 자동 등록 가능

**제한사항**
- 복잡한 데이터베이스 구조 및 실시간 동기화 어려움
- API 한도 및 속도 문제로 대규모 자동화 제한

---

### 4. **Make.com (무료 플랜)**  
**기본 제한사항**
- 월 1000회의 작업 실행 (operations)
- 최대 5개의 시나리오 동시 운영 가능
- 15분 단위 스케줄링만 지원 (실시간 반응 불가)

**자동화 가능 범위**
- Google Sheets ↔ Notion ↔ ChatGPT ↔ Slack 등 다양한 서비스 연동
- 예: “유튜브 스크립트 작성 요청 → ChatGPT 요청 → Notion DB 자동 저장” 자동화
- 데이터 수집 → 분석 → 정리 → 공유까지 자동화 가능

**제한사항**
- 작업 수 제한으로 인해 복잡하거나 대용량 자동화 어려움
- 실시간 처리 불가 → 트리거 이후 최소 15분 지연 발생

## ⚠️ 자동화 제한사항 정리

| 항목 | 제한사항 요약 |
|------|----------------|
| **비용** | 외부 API/서비스 무료 한도 초과 시 과금 발생 가능 |
| **속도** | Make.com 스케줄링 지연(최소 15분), GPT/Notion API 응답 속도에 좌우됨 |
| **복잡성** | 고급 조건문/실시간 반응/고품질 이미지/음성 자동 생성 등은 무료 플랜에서는 어려움 |
| **보안 및 안정성** | 서비스별 API 변경/정책 변화에 따른 중단 가능성 있음 |
