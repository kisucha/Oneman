## 📊 각 도구별 무료 운영 가능성 및 확장 시 비용

### 1. **Google Sheets**  
- ✅ 완전히 무료  
- 자동화 시나리오에 필요한 Apps Script, Form 연동 모두 무료 제공  
- **확장 시 비용**: 없음  

### 2. **Notion**  
- ✅ 완전히 무료 (개인 사용 기준)  
- Make.com 연동 시 API 한도 내에서는 비용 없음  
- **확장 시 비용**:  
  - API 초과 시: 유료 플랜(Professional 이상) 필요  
  - 협업 멤버 초과 시: 유료 플랜 필요  

### 3. **Make.com**  
- ⚠️ 제한적 무료 (무료 플랜: 1,000 operations/월, 최대 5 시나리오)  
- 현재 제안된 자동화 시나리오(단어 30개 기준)는 매주 1회 실행 시 약 30 operations 소모  
  → 월 120 operations로 **무료 플랜 내 충분히 운영 가능**  
- **확장 시 비용**:  
  - 운영량 증가 시: "Pro" 요금제 (~$9/월) 필요  
  - operations 초과 시 추가 비용 발생  

### 4. **ChatGPT**  
- ⚠️ 무료 제공 있지만 API 사용 시 비용 발생  
- Make.com 내 OpenAI 모듈 사용 시 API 호출 비용 부과  
  - 예: GPT-3.5-turbo 기준 약 $0.002/1,000 tokens  
  - 단어 1개당 약 50~100 tokens 소모 → 30개 기준 약 $0.003 ~ $0.006  
  → 월간 $0.01 수준으로 **거의 무료 수준이나 API 키 필요**  
- **확장 시 비용**:  
  - 대량 처리 시 OpenAI 요금제에 따라 과금  
  - GPT-4 사용 시 비용 증가
