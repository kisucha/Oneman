슬라이드 내용 (20장)

[인트로 슬라이드]
Vocabulary Training — Level 2
Today's 20 Words

──────────────────────────────
[슬라이드 1]
1. Adequate  ( /ˈædɪkwət/ )
적절한, 충분한
──────────────────────────────
We need adequate funding for this project.
우리는 이 프로젝트를 위해 적절한 자금이 필요하다.

[슬라이드 2]
2. Analyze  ( /ˈænəlaɪz/ )
분석하다
──────────────────────────────
Let's analyze the data before making a decision.
결정을 내리기 전에 데이터를 분석합시다.

[슬라이드 3]
3. Approach  ( /əˈproʊtʃ/ )
접근하다, 방법
──────────────────────────────
We need a new approach to solve this problem.
이 문제를 해결하기 위해 새로운 접근 방식이 필요하다.

[슬라이드 4]
4. Benefit  ( /ˈbenɪfɪt/ )
이익, 혜택
──────────────────────────────
Regular exercise has many health benefits.
규칙적인 운동은 많은 건강상의 이점이 있다.

[슬라이드 5]
5. Capacity  ( /kəˈpæsəti/ )
능력, 수용력
──────────────────────────────
The stadium has a capacity of 50,000 people.
그 경기장은 5만 명을 수용할 수 있다.

[슬라이드 6]
6. Concept  ( /ˈkɑːnsept/ )
개념, 생각
──────────────────────────────
The concept of time travel fascinates many people.
시간 여행의 개념은 많은 사람들을 매료시킨다.

[슬라이드 7]
7. Consistent  ( /kənˈsɪstənt/ )
일관된, 꾸준한
──────────────────────────────
He is consistent in his work.
그는 일에 있어 일관성이 있다.

[슬라이드 8]
8. Consider  ( /kənˈsɪdər/ )
고려하다
──────────────────────────────
Consider all options before deciding.
결정하기 전에 모든 옵션을 고려하십시오.

[슬라이드 9]
9. Contribute  ( /kənˈtrɪbjuːt/ )
기여하다
──────────────────────────────
Everyone can contribute to the team's success.
모두가 팀의 성공에 기여할 수 있다.

[슬라이드 10]
10. Create  ( /kriˈeɪt/ )
창조하다, 만들다
──────────────────────────────
Artists create beautiful paintings.
예술가들은 아름다운 그림을 창조한다.

[슬라이드 11]
11. Define  ( /dɪˈfaɪn/ )
정의하다
──────────────────────────────
Can you define this term for me?
이 용어를 저에게 정의해 줄 수 있나요?

[슬라이드 12]
12. Demonstrate  ( /ˈdemənstreɪt/ )
보여주다, 입증하다
──────────────────────────────
She demonstrated excellent leadership skills.
그녀는 훌륭한 리더십 능력을 보여주었다.

[슬라이드 13]
13. Efficient  ( /ɪˈfɪʃnt/ )
효율적인
──────────────────────────────
This new system is more efficient than the old one.
이 새 시스템은 구 시스템보다 더 효율적이다.

[슬라이드 14]
14. Enhance  ( /ɪnˈhæns/ )
향상시키다, 개선하다
──────────────────────────────
Good lighting can enhance the room's appearance.
좋은 조명은 방의 외관을 향상시킬 수 있다.

[슬라이드 15]
15. Evaluate  ( /ɪˈvæljueɪt/ )
평가하다
──────────────────────────────
We need to evaluate the project's progress.
우리는 프로젝트의 진행 상황을 평가해야 한다.

[슬라이드 16]
16. Evidence  ( /ˈevɪdəns/ )
증거
──────────────────────────────
The police found evidence at the crime scene.
경찰은 범죄 현장에서 증거를 발견했다.

[슬라이드 17]
17. Flexible  ( /ˈfleksəbl/ )
유연한
──────────────────────────────
Our schedule is flexible and can be adjusted.
우리의 일정은 유연하며 조정될 수 있다.

[슬라이드 18]
18. Identify  ( /aɪˈdentɪfaɪ/ )
식별하다, 확인하다
──────────────────────────────
Can you identify the main problem?
주요 문제를 식별할 수 있나요?

[슬라이드 19]
19. Implement  ( /ˈɪmplɪment/ )
이행하다, 실행하다
──────────────────────────────
We will implement the new policy next month.
우리는 다음 달에 새 정책을 이행할 것이다.

[슬라이드 20]
20. Inevitable  ( /ɪnˈevɪtəbl/ )
불가피한, 필연적인
──────────────────────────────
Change is inevitable in life.
변화는 삶에서 불가피하다.

[아웃트로 슬라이드]
Great work!
